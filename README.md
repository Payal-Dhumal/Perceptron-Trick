# Perceptron-Trick
Perceptron algorithm built from scratch using Python, demonstrating how linear classification works along with decision boundary visualization on generated data.

🚀 Perceptron from Scratch

Implementation of the Perceptron algorithm using Python and NumPy, with visualization of the decision boundary on a 2D dataset.

📌 Features
Built from scratch (no ML libraries)
Binary classification
Decision boundary visualization
⚙️ Tech Stack

Python, NumPy, Matplotlib, Scikit-learn

📊 Output
Data points plotted using scatter plot
Decision boundary shown as a line
